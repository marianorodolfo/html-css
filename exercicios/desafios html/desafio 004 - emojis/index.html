A idéia desse desafio é explorar a parte das listas e tambem inserir emojis em nosso site.

Você pode acessar o site para ver os códigos dos emojis

DICA IMPORTANTE: Acabamos de colocar acima, um link no site utilizando a tag <a> para acessar uma página externa.

Utilizamos a tag: target blank que significa que irá abrir em uma nova janela no navegador.

Utilizamos tambem a tag rel external que significa que o acesso a essa página, será feito fora do meu site.

Para adicinar emojis precisa digitar primeiro &#x (x em minúsculo) e o código do emoji, excluido U+.

Para adicionar listas ordenadas, utilizar a tag <ol>. Tem 5 opções:

  1. 1
  2. A
  3. a
  4. I
  5. i

Para adicionar listas não ordenadas, utilizar a tag <ul>. Tem 3 opções:

  1. type disc
  2. type circle
  3. type square
